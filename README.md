# Web Servicios Informáticos

Landing web para presentar servicios informáticos básicos.

## Objetivo

Crear una web sencilla, clara y profesional que sirva como práctica DAW y como carta de presentación real.

## Servicios incluidos

- Instalación de Windows.
- Limpieza y optimización de equipos.
- Copias de seguridad.
- Soporte remoto básico.
- Mantenimiento preventivo.
- Asesoramiento informático básico.

## Secciones

- Inicio.
- Servicios.
- Proceso de trabajo.
- Precios orientativos.
- Preguntas frecuentes.
- Contacto.

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- Diseño responsive
